# Класифікація автомобілів за зображеннями

Кваліфікаційна робота бакалавра — метод класифікації автомобілів 
за зображеннями засобами комп'ютерного зору.

## Технології
- Python 3.10
- PyTorch 2.0 (EfficientNet-B0)
- Tkinter (GUI)
- PostgreSQL

## Запуск
pip install -r requirements.txt
python src/app.py
